## Request Flow

```
User (Streamlit)
    │
    ▼
API Gateway :8000
    │
    ├──► POST /api/v1/query
    │         │
    │         ▼
    │    Query Service :8004  (LangGraph)
    │         │
    │    ┌────┴─────────────────────┐
    │    │  1. Planner Agent        │ ← Claude breaks question into sub-tasks
    │    │  2. Search Agent         │ ← Queries Pinecone vector DB
    │    │  3. Timeline Agent       │ ← Calls Timeline Service :8005
    │    │  4. Decision Agent       │ ← Claude analyzes dissent + risks
    │    │  5. Answer Agent         │ ← Claude generates final answer
    │    └──────────────────────────┘
    │
    └──► POST /api/v1/ingest
              │
              ▼
         Ingestion Service :8001
              │
         ┌────┴────────────────────┐
         │ Email Parser            │
         │ Meeting Parser          │
         │ Jira Parser             │
         └────┬────────────────────┘
              │
         ┌────┴───────────────────────┐
         │  Embedding Service :8002   │ ← Pinecone upsert
         │  Graph Service     :8003   │ ← Neo4j graph build
         └────────────────────────────┘
```

## Technologies Per Service

| Service | Framework | Key Libraries |
|---|---|---|
| api-gateway | FastAPI | httpx, redis |
| ingestion | FastAPI | custom parsers |
| embedding | FastAPI | LangChain, OpenAI, Pinecone |
| graph | FastAPI | neo4j async driver |
| query | FastAPI | LangGraph, Anthropic, Pinecone |
| timeline | FastAPI | Anthropic, Pinecone |
| frontend | Streamlit | httpx |
| infra | Docker | Redis, Neo4j |
